# יומן סופר סת"ם (Stam Sofer Diary) 📜

אפליקציה מתקדמת לניהול, מעקב ותיעוד עבודת סופר סת"ם.
האפליקציה מאפשרת לסופרים לנהל פרויקטים של כתיבת ספרי תורה, תפילין ומזוזות, לעקוב אחר זמני עבודה, לחשב הספקים ורווחים, ולצפות בסטטיסטיקות מפורטות.

![App Icon](assets/icon/icon.png)

## ✨ תכונות עיקריות

### ⏱️ ניהול זמן וכתיבה
*   **טיימר חכם:** מדידת זמן כתיבה מדויקת עם אפשרות לעצירה והמשך.
*   **הזנה ידנית:** אפשרות להוספת רשומות כתיבה בדיעבד (כולל תאריך ושעות).
*   **תאריך עברי:** תצוגה מלאה של התאריך העברי בכל מסכי האפליקציה.

### 📚 ניהול פרויקטים
תמיכה מלאה בשלושה סוגי פרויקטים עם לוגיקה ייחודית לכל אחד:
1.  **ספר תורה:**
    *   מעקב לפי עמודים ושורות.
    *   הגדרת מספר עמודים ושורות לעמוד.
    *   תצוגה ויזואלית (גריד) של התקדמות הכתיבה בכל עמוד (מילוי לפי אחוזים).
2.  **תפילין:**
    *   ניהול סטים (ראש + יד), או פרשיות בודדות.
    *   מעקב אחר 4 הפרשיות (קדש, והיה כי יביאך, שמע, והיה אם שמוע).
    *   חישוב אוטומטי של "זוגות" וסטים שנשלמו.
3.  **מזוזות:**
    *   מעקב לפי יחידות ושורות (לפי תקן של 22 שורות למזוזה).

### 📊 סיכומים וסטטיסטיקה
*   **סיכום יומי:** הצגת הספק יומי, זמן עבודה, רווח נקי ועמידה ביעדים.
*   **סיכום חודשי:** טבלה מרכזת של כל העבודה בחודש העברי, כולל חישוב ימי עבודה בפועל.
*   **סיכום פרויקט:** מבט על התקדמות הפרויקט כולו, כולל ממוצע דקות לשורה/פרשייה וצפי סיום.
*   **חישוב רווחים:** הזנת מחיר והוצאות לכל פרויקט וקבלת דוח רווח נקי בזמן אמת.

### ⚙️ הגדרות וגיבוי
*   שמירת נתונים מקומית ומאובטחת.
*   אפשרות לאיפוס נתונים מלא.
*   ממשק נקי ונוח לשימוש בעברית.

## 🛠️ טכנולוגיות

האפליקציה פותחה באמצעות **Flutter** ומשתמשת בספריות הבאות:
*   `kosher_dart`: לחישובי לוח שנה עברי ותאריכים.
*   `shared_preferences`: לשמירת נתונים מקומית במכשיר.
*   `url_launcher`: לפתיחת קישורים חיצוניים.
*   `flutter_launcher_icons`: ליצירת אייקונים מותאמים.

### אנדרואיד (Android)
1.  הורד את קובץ ה-`apk`.
2.  התקן אותו על המכשיר שלך (ייתכן שתצטרך לאשר התקנה ממקורות לא ידועים).

### ווינדוס (Windows)
1.  הורד את קובץ ה-`zip`.
2.  חלץ את התיקייה למחשב.
3.  הפעל את הקובץ `stamsofer.exe` מתוך התיקייה.
